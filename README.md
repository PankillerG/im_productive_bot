# im_productive_bot

### Installation
Run the following commands as root:
```bash
sudo apt update
sudo apt install git
git clone https://github.com/PankillerG/im_productive_bot.git
./im_productive_bot/setup.sh
```

### Usage
Start, stop and restart bot:
```bash
start_im_productive_bot
restart_im_productive_bot
stop_im_productive_bot
```
